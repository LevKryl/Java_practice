# Статические методы и полиморфизм

## Практика 1: Почему динамический полиморфизм не работает со статическими методами

Статические методы принадлежат классу, а не объекту. Они привязаны к классу на этапе компиляции, а не к объекту во время выполнения.

Динамический полиморфизм работает через виртуальную таблицу методов (vtable), которая существует только для нестатических методов.
