**ПРАКТИКА 2**
При отсутствии оператора break в блоках case происходит последовательное выполнение всех следующих блоков
В приведённом примере, если value равно 2, выполнится блок case 2, который присвоит переменной i значение 2. Затем, из-за отсутствия break, управление перейдёт к блоку case 3, где переменной i будет присвоено значение 3
