# Practice 2

Практическая работа по курсу «Язык программирования Java».

В данной программе мы реализуем простой ввод данных с консоли. Пользователь вводит своё имя, после чего программа выводит приветствие.

Для чтения данных используется класс Scanner.

Scanner scanner = new Scanner(System.in);

Основные действия программы:

1. Создаём объект Scanner для чтения данных из консоли.
2. Просим пользователя ввести имя.
3. Сохраняем введённое имя в переменную name.
4. Выводим приветствие с этим именем.
5. Закрываем Scanner после завершения ввода.

Пример работы программы:

Введите ваше имя: Лев
Привет, Лев!
