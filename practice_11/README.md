
## Что делает каждый спецификатор доступа

private
Члены класса с пометкой private доступны только внутри того же самого класса. Никакие другие классы не имеют к ним доступа, даже классы-наследники.

default (без спецификатора)
Если вы не пишете никакой спецификатор, это называется default или пакетный доступ. Такие члены класса доступны только внутри того же пакета. Классы из других пакетов не имеют к ним доступа, даже если они являются наследниками.

protected
Члены класса с пометкой protected доступны внутри того же пакета. Кроме того, они доступны в классах-наследниках, даже если эти наследники находятся в других пакетах.

public
Члены класса с пометкой public доступны из любого класса в любом пакете. Ограничений нет
