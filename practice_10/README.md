В результате выполнения этого кода переменная a1 начнет ссылаться на тот же массив, что и a2, то есть на массив {6,7,8,9,10}. Изначальный массив {1,2,3,4,5}, на который указывала a1, останется в памяти, но потеряет ссылку и станет доступным для сборщика мусора. Обе переменные a1 и a2 после присваивания будут ссылаться на один и тот же объект массива, поэтому изменение элемента через a1, например a1[0] = 100, приведет к тому, что при обращении через a2[0] мы также увидим значение 100. При этом размер массива, на который теперь ссылается a1, станет равен 5, но исходные элементы 1,2,3,4,5 будут утеряны для программы.
