# Переопределение метода equals()

## Соглашения для equals()

1. **Рефлексивность** - объект равен самому себе (x.equals(x) == true)

2. **Симметричность** - если x.equals(y) == true, то y.equals(x) == true

3. **Транзитивность** - если x.equals(y) и y.equals(z), то x.equals(z)

4. **Постоянство** - многократный вызов equals() дает одинаковый результат

5. **Сравнение с null** - x.equals(null) == false

## Реализация equals() в примере

1. Проверка this == obj
2. Проверка obj == null
3. Проверка getClass() == obj.getClass()
4. Приведение типа
5. Сравнение всех значимых полей
