# super и this

## Практика 1: три варианта super

1. super() - вызов конструктора родителя
2. super.x - доступ к полю родителя
3. super.print() - вызов метода родителя

## Практика 2: ответ

При вызове method() объекта класса C:
- super.a обратится к переменной a из класса A
- super.method() вызовет метод из класса A
- Переменная a и метод method() наследуются из A в B и C

## Практика 3: this()

this() вызывает другой конструктор того же класса. Должен быть первым.
