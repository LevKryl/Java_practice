В примере есть базовый абстрактный класс Shape, который описывает геометрическую фигуру. Он содержит абстрактный метод getArea() для вычисления площади.
Класс Rectangle наследуется от Shape. Он реализует метод getArea() как произведение ширины на высоту.
Класс Square также наследуется от Shape. Он реализует метод getArea() как квадрат стороны.
Принцип подстановки Лисков соблюдается, так как Rectangle и Square могут заменять Shape без ошибок. Квадрат не наследует прямоугольник, что предотвращает нарушение ожидаемого поведения.
