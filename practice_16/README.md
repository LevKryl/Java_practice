# Перегрузка методов (overloading)

## Что выведет программа

В цикле for (1):
Number=1
Number=11
Number=1.11
Number=11.11

Прямые вызовы (2):
Integer = 1
int = 11
Float = 1.1100
Number=11.11

## Почему результаты разные

В цикле переменная n имеет тип Number. Компилятор выбирает метод printNum(Number n), потому что тип ссылки - Number.

При прямых вызовах компилятор видит конкретные типы аргументов и выбирает соответствующие методы.
