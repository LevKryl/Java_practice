## this и super

**this** - ссылка на текущий объект
**super** - ссылка на объект родительского класса


## ПРАКТИКА 2
**private (a4)** - не виден в B и C
Объявлен в A, но классы B и C не могут его использовать.

**default (a1)** - виден в B и C
Работает, потому что все классы в одном пакете.

**protected (a3)** - виден в B и C
Доступен во всех подклассах, даже в других пакетах.

**public (a2)** - виден в B и C
Доступен везде.

## Вывод

В классах B и C можно использовать a1, a2, a3, но нельзя a4.
